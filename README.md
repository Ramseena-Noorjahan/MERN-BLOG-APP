# MERN Stack Blog Project

This is a MERN (MongoDB, Express.js, React.js, Node.js) stack blog project. It allows users to create, read, update, and delete blog posts. The project is hosted on Vercel.

## Frontend

The frontend of this project is built using React.js. It utilizes the following packages:

- `react-quill`: A rich text editor component for React.
- `react-router-dom`: Provides routing functionality for React applications.
- `react-scripts`: Tools and scripts for React development.
- `react-time-ago`: A simple and efficient way to display formatted timestamps.

## Backend

The backend of this project is built using Node.js and Express.js. It interacts with a MongoDB database using Mongoose for object modeling.

The backend utilizes the following packages:

- `bcryptjs`: Library for hashing passwords.
- `cors`: Middleware for enabling CORS (Cross-Origin Resource Sharing).
- `dotenv`: Loads environment variables from a `.env` file into `process.env`.
- `express`: Web framework for Node.js.
- `express-fileupload`: Middleware for handling file uploads.
- `jsonwebtoken`: Implements JSON Web Tokens for authentication.
- `mongoose`: An Object Data Modeling (ODM) library for MongoDB and Node.js.
- `uuid`: Generates UUIDs (Universally Unique Identifiers).

## Deployment

This project is hosted on [Vercel](https://vercel.com/). Vercel is a cloud platform for static sites and Serverless Functions that fits perfectly with frontend React applications, providing easy deployment and scalability.

## Getting Started

To get started with this project, follow these steps:

1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Install dependencies for both frontend and backend using `npm install`.
4. Set up environment variables as needed.
5. Start the frontend and backend servers using `npm start`.
6. Access the application in your browser.
